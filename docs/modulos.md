# Módulos do Sistema F1A

---

# 1. ADM Cartão

Responsável pelo controle operacional de maquinetas.

Funcionalidades:

- Cadastro de maquinetas
- Controle de estoque
- Entrada de equipamentos
- Saída de equipamentos
- Cadastro de fornecedores
- Consulta de movimentações

---

# 2. Contratos

Responsável pelo gerenciamento contratual.

Funcionalidades:

- Cadastro de contratos
- Alteração contratual
- Controle de vigência
- Controle de renovação
- Consulta de contratos

---

# 3. CRM

Responsável pela gestão comercial.

Funcionalidades:

- Cadastro de leads
- Pipeline de vendas
- Registro de contatos
- Oportunidades
- Histórico comercial

---

# 4. Financeiro

Responsável pela gestão financeira.

Funcionalidades:

- Contas a receber
- Contas a pagar
- Fluxo de caixa
- Relatórios financeiros
- Conciliação

---

# 5. Pagamentos

Responsável pelo controle dos recebimentos.

Funcionalidades:

- Registro de pagamentos
- Controle de vencimentos
- Controle de inadimplência
- Baixas financeiras
- Relatórios

---

# 6. Visitas

Responsável pelo acompanhamento externo.

Funcionalidades:

- Agenda de visitas
- Registro de visitas
- Histórico de visitas
- Relatórios de visitas
- Planejamento comercial

---

# 7. Administração

Responsável pela configuração do sistema.

Funcionalidades:

- Cadastro de usuários
- Perfis de acesso
- Controle administrativo
- Auditoria
- Configurações gerais

---

# 8. Dashboard

Responsável pela visualização gerencial.

Funcionalidades:

- Indicadores
- Gráficos
- Resumos operacionais
- KPIs
- Alertas
